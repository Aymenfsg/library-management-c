# Library Management System in C

A command-line application to manage a book library, built in C.

## Features
- Add books manually (title, author, year)
- Display all books
- Delete a book by ID
- Data stored using a singly linked list

## Concepts used
- Structs
- Pointers
- Linked lists
- Dynamic memory allocation (malloc, free)

## How to compile and run
```bash
gcc library.c -o library.exe
./library.exe
```
